Contexte du projet

Réalisé lors de ma première année de BUT Réseaux & Télécommunications, ce projet avait pour but de maîtriser les fondamentaux du développement web. Le défi était de concevoir un site web complet sans utiliser de constructeurs de pages (CMS) ni de frameworks, en écrivant 100% du code à la main.

En équipe de trois, nous avons choisi de créer une plateforme regroupant nos passions respectives : Les séries et les Mangas.

Approche Technique & Collaboration

Ce projet a été l'occasion d'apprendre à structurer une page web et à la styliser proprement :

  • Structure Sémantique (HTML5) : Utilisation des balises modernes (header, nav, section, article, footer) pour assurer une bonne hiérarchie du contenu.
  • Design & Mise en page (CSS3) : Création d'une identité visuelle cohérente malgré les thèmes différents. Utilisation de Flexbox pour la mise en page et les menus de navigation.
  • Travail d'équipe : Répartition des tâches par "pages thématiques" et unification du code pour assurer une navigation fluide entre les différentes sections du site.
